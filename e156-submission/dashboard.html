◉ Project Overview

The ROBMA Reproducibility Capsule is a thin, reviewer-auditable package that wraps example data, a narrated tutorial walkthrough, and deterministic validation checksums around a robust Bayesian model-averaging (RoBMA) workflow.

The contribution is packaging and reproducibility support, not a reimplementation of the RoBMA R package. Reviewers receive a fixed example input, a rerun script, and manuscript-ready materials so they can audit the analysis without reconstructing it from scratch.

Target journal: F1000Research (Software Tool Article).
